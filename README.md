AIM : STUDY OF NUMPY LIBRARY THEORY
: NumPy stands for Numerical Python. It is a powerful Python library used for mathematical and numerical computations, especially when working with arrays and matrices. It is one of the most important libraries used in Data Science, Machine Learning, Artificial Intelligence, and Scientific Computing.

1.Importing NumPy Library Algorithm: Start the program. Import the NumPy library using import numpy as np. Use np as an alias to access NumPy functions.

2️.Creating NumPy Arrays Algorithm: Define a 1D array using np.array([elements]). Define a 2D array using nested lists inside np.array([[...],[...]]). Store arrays in variables (a and b). Print the arrays. Purpose: To create and display one-dimensional and two-dimensional arrays.

3️.Finding Dimension of Array Algorithm: Use .ndim attribute on the array. Print the result. For 1D array → output is 1. For 2D array → output is 2. Purpose: To determine how many axes (dimensions) the array has.

4️.Finding Size of Array Algorithm: Use .size attribute on the array. Print the total number of elements in the array. Purpose: To calculate total elements present in the array.

5️.Finding Shape of Array Algorithm: Use .shape attribute on the array. It returns a tuple representing (rows, columns). Purpose: To determine the structure of the array.

6️.Finding Data Type of Array Algorithm: Use .dtype attribute on the array. It returns the data type of elements stored in the array. Purpose: To check the type of elements (int, float, etc.).

7️.Creating Arrays Using Built-in Functions A) Using np.zeros() Algorithm: Call np.zeros((rows, columns)). It creates an array filled with zeros. B) Using np.ones() Algorithm: Call np.ones((rows, columns)). It creates an array filled with ones. C) Using np.arange() Algorithm: Use np.arange(start, stop, step). It generates numbers starting from start. Stops before stop. Increments by step. D) Using np.linspace() Algorithm: Use np.linspace(start, stop, number_of_values). It divides the range into equal intervals. Returns evenly spaced values between start and stop.

8️.Arithmetic Operations on Arrays Algorithm: Perform element-wise operations. For multiplication → b * 2 For addition → a + 5 NumPy automatically applies operation to each element. Purpose: To perform vectorized operations efficiently.

9️. Statistical Functions in NumPy A) Mean Algorithm: Use np.mean(array). Add all elements. Divide by total number of elements. Return average value. B) Median Algorithm: Use np.median(array). Sort the array. If odd → return middle value. If even → return average of two middle values. C) Maximum Algorithm: Use np.max(array). Compare all elements. Return the largest value. D) Minimum Algorithm: Use np.min(array). Compare all elements. Return the smallest value.

CONCLUSION : NumPy enables efficient numerical computation and is widely used in Data Analysis and EDA (Exploratory Data Analysis).
